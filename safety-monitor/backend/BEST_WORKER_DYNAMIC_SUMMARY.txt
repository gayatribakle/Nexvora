3. API INTEGRATION
============================================================

ENDPOINT 1: GET /api/analytics/safety-scores
Purpose: Get top 20 workers by safety score
Returns: Array of {worker_id, worker_name, department, safety_score, total_violations}

ENDPOINT 2: GET /api/workers?limit=1000
Purpose: Get all workers for fine data and employee IDs
Returns: {workers: [{id, employee_id, total_fine_amount, photo_url, status}, ...]}

DATA MAPPING:
1. Fetch safety scores (top 20)
2. Fetch all workers (for additional data)
3. Match workers by ID
4. Create WorkerData objects with:
   - id: W001, W002, etc. (formatted)
   - name: From safety_scores.worker_name
   - photo: From workers.photo_url
   - behaviourScore: From safety_scores.safety_score
   - totalFine: From workers.total_fine_amount
   - monthlyViolations: From safety_scores.total_violations
   - department: From safety_scores.department
   - employee_id: From workers.employee_id

5. Sort by behaviourScore (descending)
6. Set top 5 for leaderboard
7. Set #1 as best worker
8. Calculate average score
